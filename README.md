# Prompts

## Executive Summary

This repository serves as a comprehensive collection of specialized AI prompt templates designed across multiple domains, featuring role-based expertise definitions for AI systems. Each template is carefully crafted to enable AI assistants to adopt specific professional personas with domain-specific knowledge, communication styles, and best practices.

The collection provides structured frameworks that define expert roles, responsibilities, and interaction patterns, enabling consistent and high-quality AI-powered assistance across various specialized fields including finance, law, healthcare, technology, and more.

**Key Features:**
- **Role-Based Expertise**: Templates define specific professional roles with detailed competency frameworks
- **Domain Coverage**: Spans multiple industries and specializations
- **Structured Approach**: Consistent template format for easy adaptation and deployment
- **Best Practices**: Incorporates industry standards and professional guidelines
- **Customizable**: Templates can be adapted to specific organizational needs

## How to Use

Getting started with these prompt templates is simple:

1. **Copy the Prompt**: Navigate to the desired prompt template and copy its contents
2. **Paste to Your GenAI**: Paste the prompt into your preferred Generative AI platform (ChatGPT, Claude, Gemini, etc.)
3. **Update the DATA_TEMPLATE**: If using a template with `[DATA_TEMPLATE]`, replace the placeholder data with your specific information
4. **Paste Again**: Copy the updated data and paste it to the GenAI to provide context
5. **Start Interacting**: Your AI assistant is now configured with the specialized role and ready to help

**Example Workflow:**
```
1. Copy "French Wealth and Financial Advisor" prompt → Paste to GenAI
2. Open "French Wealth and Financial Advisor [DATA_TEMPLATE]"
3. Replace template data with your client's actual information
4. Paste the customized data to the GenAI
5. Begin your financial advisory conversation
```
